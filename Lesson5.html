1. Что такое < ol >, < ul > и < li >?

ol - ordered list - упорядоченный список, с порядковыми номерами элементов.
ul - unordered list - неупорядоченный список, по умолчанию с точками перед элементами.
li - list item - элемент списка, может быть только внутри тегов ul или ol

7. Как убрать подчеркивание для всех ссылок на странице?

Прописать в теге style или внешнем CSS-файле "a {text-decoration: none}" Если CSS-файл подключен к нескольким страницам, то можно за каскадом "page.a {text-decoration: none}"

10. Перечислить по пять знакомых вам псевдоклассов и псевдостилей с указанием их функционала.

Псевдостили
  • :hover - задать стиль при наведении
  • :active - стили ссылки в момент нажатия
  • :visited - стили посещённой ссылки
  • :first-letter - задать стили первой буквы абзаца
  • :first-line - задать стили первой строки абзаца
Псевдоклассы
  • :first-child, :last-child - стили для первого или последнего элемента внутри родителя
  • :first-of-type, :last-of-type - стили для первого или последнего элемента определённого типа внутри родителя
  • :not - стили для элементов, в котором нет указанного селектора
  • :checked - задать по умолчанию выбранный переключатель или флажок
  • :disabled - заблокировать элемент формы
